# Memory Match Master

A classic 4x4 memory matching card game built with Angular. Test your memory by finding all matching pairs of cards!

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the application:
   ```bash
   npm run dev
   ```
   Open your browser and navigate to `http://localhost:4200/` to play the game.
